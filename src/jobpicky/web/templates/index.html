调整岗位雷达

连接飞书

将推荐岗位同步到你的飞书多维表格，并每日更新新增岗位。

第 1 步，共 9 步

打开飞书开放平台

    准备应用

    打开飞书开放平台

    创建一个仅属于你自己的飞书应用，用来将 JobPicky 推荐岗位同步到你的多维表格。

    打开飞书开放平台https://open.feishu.cn/app

    JobPicky 不需要，也不会要求你提供飞书账号密码。

    创建应用

    创建企业自建应用

    1. 点击“创建企业自建应用”。
    2. 应用名称填写 JobPicky。
    3. 描述填写“同步个性化推荐岗位到飞书”。
    4. 图标可暂时使用默认值，然后点击创建。

    应用凭据

    获取 App ID 和 App Secret

    进入“凭证与基础信息”，复制 App ID 和 App Secret。Secret 只保留在当前页面会话中，刷新后不会恢复。

    API 权限

    开通多维表格 API 权限

    在“权限管理”中选择应用身份，开通与当前同步调用匹配的最小权限：

    查看、评论、编辑和管理多维表格bitable:app
    搜索不到权限怎么办

    确认身份类型选择“应用身份”，搜索权限 key。若企业策略要求发布应用或管理员审核，请完成对应流程后再继续；这不是所有用户都必须执行的固定步骤。

    JobPicky 不需要通讯录、机器人或消息权限。

    应用发布

    版本管理与发布

    创建应用后，进入“版本管理与发布”，填写更新说明并保存，再提交发布申请。应用发布后，当前配置和权限才会生效。

    1. 打开“版本管理与发布”。
    2. 填写更新说明并点击“保存”。
    3. 在确认弹窗中点击“确认发布”。

    准备多维表格

    新建一份空白多维表格

    如果你还没有要同步的 Base，按右侧截图新建一份空白多维表格。红框标出了每一步要点击的位置，最后一张是创建完成后的空白表格。

    选择你的情况

    选择后，这里会显示对应操作。

    创建一份空白多维表格

    1. 打开飞书多维表格官网并点击“立即体验”。
    2. 进入多维表格工作台后，点击“多维表格”。
    3. 在模板中心选择“新建空白多维表格”。
    4. 创建完成后保持这个 Base 打开。
    打开飞书多维表格官网https://www.feishu.cn/product/base

    创建完成后保持这个 Base 打开，下一步会把 JobPicky 添加为文档应用。

    打开要同步的多维表格

    1. 在飞书中打开你准备使用的 Base。
    2. 确认你可以编辑该 Base。
    3. 保持页面打开，下一步添加 JobPicky 文档应用。

    可以使用已有 Base；JobPicky 只管理“求职工作台”及其同步字段，不会要求你手动建表。

    文档权限

    将 JobPicky 添加为文档应用

    1. 打开多维表格右上角“...”，进入“更多”。
    2. 点击“添加文档应用”，搜索 JobPicky。
    3. 将权限设为“可管理”，再点击“添加”。
    搜索不到 JobPicky 应用？
    • 尚未开通任何多维表格 API 权限；
    • 权限尚未发布或审核；
    • 当前登录企业与创建应用的企业不一致。

    准备链接

    复制 Base 链接

    回到刚刚打开的多维表格，复制浏览器地址栏中包含 /base/ 的链接。不要复制 /wiki/ 链接。

    写入目标

    记录会写入 JobPicky 自动创建或接管的“求职工作台”，不是刚打开 Base 时看到的初始“数据表”。完成后请点击结果页的“打开飞书工作台”进入正确的表。

    同步范围

    首次连接会写入当前推荐；以后扫描会更新对应记录，不会重复创建已成功的岗位。

    断开飞书连接

    这只会停止后续同步，不会删除飞书中的工作台或已经同步的岗位。
