# v0.4.0

## 主要更新

- 重构岗位数据模型与提取流程，支持从招聘公告中拆分结构化岗位，并补充岗位分类、地点和组织信息。
- 升级匹配与推荐逻辑，提供更清晰的匹配证据，并区分校招与实习岗位，提升推荐准确性。
- 改进 WonderCV 多岗位提取和岗位种子数据，补充近期岗位与官方链接。
- 重设计岗位雷达与扫描流程，加入液态玻璃视觉、推荐视图切换、折叠偏好标签和更稳定的扫描状态。
- 修复最近推荐筛选、扫描进度和重新匹配等问题。

## 发布验证

- 本地完整测试套件通过。
- GitHub Actions 将在标签推送后执行 Python 3.11/3.12 跨平台测试、构建和 PyPI 发布。
